# Three.js Demo Collection

Proyecto académico que integra múltiples ejemplos de Three.js en una interfaz web moderna y minimalista.

## Demos incluidas

- Geometry / Minecraft
- Controls / Map
- Controls / Orbit
- Controls / Pointer Lock

## Tecnologías utilizadas

- Three.js
- HTML5
- CSS3
- UIverse Components

## Características

- Diseño minimalista
- Interfaz responsive
- Navegación mediante cards
- Implementación modular de demos
